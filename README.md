# Treinamento Neeo (Nuxt 4 & PHP)

Bem-vindo ao repositório oficial da plataforma de Treinamento Neeo! Este projeto contém a nova e atualizada plataforma de aprendizado dedicada a capacitar desenvolvedores nas trilhas de **Nuxt 4** e **PHP**.

## 🚀 O Projeto

Esta plataforma interativa oferece:
- **Trilhas Específicas:** Cursos e desafios focados em Nuxt 4 e PHP.
- **Simulador Interativo:** Desafios técnicos baseados em cenários reais para testar e aprimorar suas habilidades.
- **Interface Moderna:** Desenvolvida com Nuxt 4 e Nuxt UI para a melhor experiência de usuário.

## 🛠️ Tecnologias Utilizadas

- **Nuxt 4**
- **Vue 3**
- **Nuxt UI** & **Tailwind CSS**
- **TypeScript**

## 💻 Como Executar o Projeto Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/tiagomdss/treinamentoNuxt4.git
   ```

2. **Entre na pasta do projeto:**
   ```bash
   cd treinamentoNuxt4 
   # ou o nome configurado na sua máquina (ex: treinamento_Neeo)
   ```

3. **Instale as dependências:**
   ```bash
   pnpm install
   ```

4. **Execute o servidor de desenvolvimento:**
   ```bash
   pnpm dev
   ```

A aplicação estará disponível em `http://localhost:3000`.
